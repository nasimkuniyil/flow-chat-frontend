# FLOW CHAT MESSENGER

Flow Chat is a high-performance, real-time communication platform built with the MERN stack (Next.js, Node.js, Express, and MongoDB). It focuses on seamless user experience, instant messaging, and secure authentication.

## Features

- Real-time Messaging: Instant message delivery and receipt using WebSockets.
- Modern UI/UX: A clean, responsive interface built with Next.js and Tailwind CSS.
- Secure Authentication: User signup, login, and email verification.
- Email Notifications: Transactional emails and alerts powered by Resend.
- Status Indicators: Real-time "Online/Offline" status and typing indicators.

## Tech stack

### Frontend

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **Real-time Communication**: Socket.io-client

### Backend

- **Server**: Node.js & Express.js
- **Database**: MongoDB (with Mongoose)
- **Communication**: Socket.io
